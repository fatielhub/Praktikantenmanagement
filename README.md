# Praktikantenmanagement – Universitätspraktikum

## Projektbeschreibung
Dieses Webprojekt dient zur Verwaltung von Praktikanten innerhalb einer Fakultät.  
Benutzer können Praktikanten hinzufügen, bearbeiten, löschen und ihre Daten einsehen.  
Das Projekt enthält ein einfaches Dashboard zur Übersicht der Praktikanten.

## Technologien
- **Backend & Frontend:** Laravel 10 mit Blade Templates  
- **Datenbank:** MySQL  
- **API:** REST API, Laravel Sanctum für Authentifizierung

## Funktionen
1. **Praktikantenverwaltung**
   - Praktikanten hinzufügen, bearbeiten und löschen
   - Einsicht in die Details der Praktikanten

2. **Dashboard**
   - Übersicht aller Praktikanten
   - Statistiken pro Abteilung oder Seance
